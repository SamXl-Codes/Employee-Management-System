CSV File Format

Your CSV file must include the following columns in this order:

name,email,phone,department,role,salary,date_joined

Column Descriptions:

  • name: Employee full name (required)
  • email: Unique email address (required, valid format)
  • phone: Phone number (optional, but recommended)
  • department: Department name - must match existing department (required)
  • role: Job role title - must match existing role (required)
  • salary: Annual salary in dollars (required, numeric)
  • date_joined: Join date in YYYY-MM-DD format (required)
